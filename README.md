# learn-spring-cloud

#### 项目介绍
1. SpringBoot
2. Eureka 服务注册发现组件
3. Feign    服务调用负载均衡
4. Hystrix 断路器
5. Gateway  路由网关
6. SpringCloudConfig 分布式配置中心
7. SpringCloudBus   消息总线
8. RabbitMq 消息队列
